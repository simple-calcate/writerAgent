# 网文写作助手

桌面端网文写作辅助工具。核心理念：**不替你写，只帮你写得更好**。

自动分析章节中需要优化的片段，实时预览润色效果，保留原文可追溯。

## 功能一览

### AI 润色

- **自动检测** — 点击「AI 润色」，自动扫描全章找出薄弱片段
- **实时预览** — 点击建议卡片，编辑器直接显示润色后的效果
- **一键采纳/忽略** — 确认后替换原文，不满意可撤回
- **原文追溯** — 每处润色记录原因和原文，支持 Ctrl+Z 撤销

### 版本管理

- **自动保存** — 编辑后 1 秒自动保存
- **版本快照** — 点击「存版本」手动创建节点，或自动记录
- **版本历史** — 查看所有历史版本，随时恢复
- **版本删除** — 清理不需要的历史记录

### 文本格式化

8 种一键格式化工具：

| 工具 | 说明 |
|------|------|
| 去多余空格 | 删除行首行尾空格，合并连续空格 |
| 去空行 | 删除连续空行，保留段落间隔 |
| 中英文标点 | 英文标点转中文标点 |
| 段首缩进 | 每段开头加两个全角空格 |
| 去段首缩进 | 删除段首缩进 |
| 引号规范化 | 直引号转弯引号 |
| 数字转中文 | 阿拉伯数字转中文（0-10）|
| 句尾统一 | 确保每段以句号结尾 |

### 项目管理

- 多项目 / 多章节结构
- 双击重命名项目和章节
- 自动保存，数据持久化

### 数据存储

- 所有数据保存在本地 JSON 文件
- 设置中可查看存储路径
- 支持打开文件夹、修改保存位置

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发模式
npm run dev

# 构建生产版本
npm run build
```

## API 配置

启动后点击右上角「设置」，填入：

| 字段 | 说明 | 默认值 |
|------|------|--------|
| API Key | 你的 API 密钥 | - |
| Base URL | API 端点地址 | `https://api.openai.com/v1` |
| 模型 | 模型名称 | `gpt-4o-mini` |

支持任何 OpenAI 兼容接口：
- OpenAI（GPT-4o / GPT-4o-mini）
- DeepSeek
- Moonshot (Kimi)
- 通义千问
- 其他兼容接口

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+S` | 手动保存 |

## 技术栈

- **Electron** — 桌面端框架
- **React 18** + **TypeScript** — 前端
- **Vite** (electron-vite) — 构建工具
- **Zustand** — 状态管理
- **Tailwind CSS** — 样式
- **OpenAI SDK** — LLM 调用
